# CareerMind AI

AI-powered career intelligence system that delivers personalized, scientifically validated career reports.

## Project Structure

- `backend/`: FastAPI backend providing REST APIs, Google Sheets integration, and AI report generation.
- `frontend/`: React + Vite + TypeScript frontend for the multi-step career assessment form.

## Tech Stack

- **Backend**: Python FastAPI, SQLite, SQLAlchemy, Resend (Email), Google Sheets API.
- **Frontend**: React, Vite, TypeScript, Tailwind CSS, Axios.

## Assessment Modules

1. **RIASEC** (Holland Code)
2. **MBTI** Personality
3. **Big Five**
4. **Multiple Intelligence**
5. **Career Matching**
